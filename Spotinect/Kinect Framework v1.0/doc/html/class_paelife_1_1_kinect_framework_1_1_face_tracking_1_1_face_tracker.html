TimeSpan Paelife.KinectFramework.FaceTracking.FaceTracker.StabilizingInterval
getset

When tracking the face of the user, it may occur that, only in a particular frame, an error occurs and the face can't be detected. This way, by setting a interval where the result of the face tracking must be always the same, we garantue a more accurate results. [see more bellow]

Note that this class doesn't guarantee this property by itself. It is the implementations of this abstract class that should check for this value before notifying that the properties have changed.

The value is set to 0.5 seconds, by default.
